# Frontend — Todo App

Интерфейс списка задач: 
  Фон: тёмно-серый, почти чёрный (#0d0d0f, #16161a, #1c1c21)
  Акцент: оранжевый (#f97316) — кнопки, чекбоксы, фокус, логотип.
  Текст: светлые оттенки серого для контраста.
  Эффекты: лёгкое свечение оранжевого, плавные переходы, тени, градиенты на кнопках и иконке.

## Стек

- **React 18** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **CSS Modules** — стили компонентов
- CSS-переменные для темы (цвета, отступы, тени)

## Структура

```
src/
  api/           # Клиент API (запросы к бэкенду)
  components/    # Переиспользуемые компоненты (Layout, Header, TaskForm, TaskItem, TaskList)
  pages/         # Страницы (TodoPage)
  styles/        # Глобальные стили и переменные темы
  types/         # TypeScript-типы (Task, CreateTaskInput)
  App.tsx
  main.tsx
```

## Сборка

- `npm run build` — сборка в `dist/`.
- `npm run preview` — просмотр собранной версии.
